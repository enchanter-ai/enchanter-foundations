# Playbook — CC1.4 Commitment to competence

- **Control ID:** CC1.4
- **Control description:** The entity demonstrates a commitment to attract, develop, and retain competent individuals in alignment with objectives.
- **Evidence-collection mechanism:** Daily hash of `shared/conduct/skill-authoring.md` (codifies skill-authoring discipline) plus periodic `metric` event recording results of the 9/10-dispatches discovery test for new skills.
- **Producer (today):** `shared/conduct/skill-authoring.md`; new-skill discovery-test workflow (planned — currently manual).
- **Retention rule:** 1 year.
- **Who reviews:** Repo maintainer weekly.
- **Escalation path:** If skill-authoring.md changes without a corresponding PR review log, flag in dashboard. If a shipped skill fails the 9/10 discovery test, block merge.
- **Type II evidence shape:** Stable hash + ≥1 new-skill discovery-test result per quarter showing controls operating effectively.
- **External dependency:** None.
